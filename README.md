# Portfolio Data Management

This directory contains the structured data for the portfolio website.

## Files

- **[portfolio-data.json](file:///C:/Users/User/.gemini/antigravity/scratch/portfolio/portfolio-data.json)**: The main database for your portfolio. Edit this file to update your personal info, skills, experience, and projects.

## How to Update

1. Open `portfolio-data.json`.
2. Update the values in the JSON fields.
3. Save the file.

The website (once connected) will automatically read from this JSON file to display your latest information.
